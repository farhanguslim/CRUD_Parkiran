# Aplikasi Parkiran - Java CRUD

Aplikasi ini merupakan sistem **Manajemen Parkiran** berbasis **Java** yang menerapkan konsep  
**CRUD (Create, Read, Update, Delete)**.

Project dapat dijalankan dalam **dua mode**:
- **CLI (Console)** menggunakan `Main.java`
- **GUI (Desktop)** menggunakan `LoginFrame.java`

Aplikasi terhubung ke **Database MySQL atau Oracle** menggunakan **JDBC**.
